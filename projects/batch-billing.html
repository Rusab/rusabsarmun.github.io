CV Based Batch-Billing System

View Paper

Overview

We propose a computer vision-based billing system for the check-out of retail items on the cash counter. This innovative system aims to revolutionize the traditional barcode scanning process by enabling simultaneous detection of multiple products, significantly reducing checkout times and improving customer experience.

Technologies Used

YOLO Computer Vision ArUco Markers GUI Development Deep Learning

Key Features

  • Simultaneous detection of multiple retail products
  • GUI-based checkout application
  • YOLO-based object detection architecture
  • Integration of Hybrid ArUco Markers for weight-dependent products
  • Real-time product detection and billing

Implementation Details

The system uses a webcam placed above the checkout counter to scan retail items. We trained our model on 25 local retail products and 5 weight-dependent products using both authentic and synthetic images. A significant innovation in our approach is the use of Hybrid ArUco Markers instead of traditional QR codes for weight-dependent products, solving the long-distance detection challenge.

Results and Impact

Our system has achieved reasonable real-time retail product detection accuracy, making it viable for implementation in actual billing systems. This technology paves the way for a fully automated supermarket experience, potentially reducing checkout times and improving overall customer satisfaction in grocery stores.
